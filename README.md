Initial Setup: <br>
-> npm create vite@latest ./ 
The above command is used to install the latest vite in the current folder.

Then choose the following:
-> React
-> TypeScript
Run the following:
-> npm install
-> npm run dev

*NOTE: src/App.tsx file will be the primary router component.
